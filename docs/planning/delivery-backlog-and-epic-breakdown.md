# 11. Delivery Backlog / Epic Breakdown

## 11.1 Epic order

1. Foundation (service + DB + queue)

2. Domain core entities

3. Policy framework

4. Integration adapters

5. Hermes orchestration contracts

6. Execution pipeline + audit

7. Safe pause and rollback

8. Reporting and pilot readiness

## 11.2 Task granularity

Implement in vertical slices that provide executable behavior at every stage.

## 11.3 MVP completion criteria

- one project can be onboarded

- one goal can be ingested and decomposed

- pings/escalations and safe-pause work

- rollback from one action is possible

- reports visible in Notion
